iSnapshot 3.1.0 – Screenshots with various options

iSnapshot K'ed Utilities
 by NMac Ked


iSnapshot (was Screen Capture X) is a fast, user-friendly and efficient utilities for screenshot.

 

iSnapshot Features:

  • Supports three capture modes(Full screen, Window, Selection)
  • Supports capture the cursor within full screen capture mode
  • Supports delay capture within full screen capture mode
  • Supports window snapshot excluding shadow in window capture mode
  • Supports capture toggle between mouse selection within selection capture mode
  • Supports capture to PNG, PDF,JPG,BMP,TIFF,GIF file
  • Supports email
  • Lion compatibility

 

Note: This application contains in-app and/or external module purchases.

 

WHAT’S NEW

Version 3.1.0:

  • Save snapshot directly to Picture folder
  • Export/Merge to PDF format
  • Fixed some bugs

 

REQUIREMENTS

  • Intel, 64-bit processor
  • OS X 10.7 or later
